## 2017.11.23开始使用git更新
### 提交代码
进入到代码所在本地文件夹
git add .
git commit -m "提交说明"
git push origin branchname(origin表示远程代码仓库、branchname表示分支名，本仓库的分支为hellocpp)
### 下拉代码
进入到代码所在本地文件夹，然后
git pull origin branchname(origin表示远程代码仓库、branchname表示分支名)
